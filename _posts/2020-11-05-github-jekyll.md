简单介绍如何使用 jekyll 便捷开发个人博客

### 安装

```shell
$ sudo apt install ruby ruby-dev
$ gem install jekyll bundler
$ keyll new my-awesom-site
$ cd my-awesome-site
$ bundle install
$ bundle exec jekyll serve
```
